## ⚠️ IMPORTANTE - Información Sensible

**NUNCA commitear información sensible en archivos de esta carpeta**, como:

- ❌ Contraseñas de bases de datos
- ❌ Tokens de API
- ❌ URLs con credenciales embebidas
- ❌ Secrets de JWT
- ❌ Claves privadas
- ❌ Información personal sensible

### 🔒 Buenas prácticas:

1. **Para archivos con información sensible**: Usa la extensión `.local.*`
   - Ejemplo: `enlaces.local.md`, `credenciales.local.txt`
   - Estos archivos están en `.gitignore` y no se subirán a GitHub

2. **Para compartir configuraciones**: Usa placeholders
   - ✅ `PASSWORD=<TU_PASSWORD>`
   - ❌ `PASSWORD=MiPasswordReal123`

3. **Para URLs de producción**: Ofusca los dominios reales
   - ✅ `https://<TU_APP>.netlify.app`
   - ❌ `https://realprint.netlify.app`
