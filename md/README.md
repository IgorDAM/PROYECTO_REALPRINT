# md - Documentacion Organizada

Esta carpeta agrupa los documentos Markdown por tipo para evitar archivos sueltos en la raiz.

## Inicio rapido

- Ver indice maestro: [INDEX_GLOBAL](INDEX_GLOBAL.md)
- Ver onboarding guiado: [INDEX_ONBOARDING](INDEX_ONBOARDING.md)

## Estructura

- `00_inicio/`
  - Punto de entrada y guia inicial.
- `01_indices/`
  - Indices y mapas de documentacion.
- `02_analisis_y_valoracion/`
  - Analisis, resumen ejecutivo, valoraciones y conclusiones.
- `03_planes_y_roadmap/`
  - Planes de accion, roadmap y mejoras propuestas.
- `04_ejecucion_pasos/`
  - Registro por paso completado (`PASO_1` ... `PASO_8`).
- `05_guias_y_referencias/`
  - Guias de instalacion y referencias tecnicas.
- `06_seguridad_y_calidad/`
  - Documentos de seguridad y calidad de codigo.
- `07_app_realprint/`
  - Documentacion funcional/tecnica originalmente ubicada en `App-RealPrint/`.
- `99_registro/`
  - Registro historico y cambios de documentacion.

## Nota

Se mantiene `README.md` en la raiz del repositorio por convencion.
